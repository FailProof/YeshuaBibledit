Using Google Translate in Bibledit - Developer Notes
====================================================

One method to use Google Translate is through the Google Cloud Platform C++ Client Libraries.
Information: https://github.com/googleapis/google-cloud-cpp
While building this through $ cmake the build machine started to freeze up.
It then indicated that there's was no application memory left.

Another way of using Google Translate is through the gcloud CLI.
Information: https://cloud.google.com/sdk/gcloud/
Installing gcloud: https://cloud.google.com/sdk/docs/install

Activate gloud: $ gcloud auth activate-service-account --key-file=key.json

Do the translation:

$ export GOOGLE_APPLICATION_CREDENTIALS=`pwd`"/key.json"
$ curl -s -X POST -H "Content-Type: application/json" \
    -H "Authorization: Bearer "$(gcloud auth application-default print-access-token) \
    --data "{
  'q': 'The quick brown fox jumps over the lazy dog',
  'source': 'en',
  'target': 'fr',
  'format': 'text'
}" "https://translation.googleapis.com/language/translate/v2"
